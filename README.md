Single page portfolio website for Authur Ramesh Rishvaanth, an AI / ML engineer based in Chennai.  
The site is built to showcase experience, skills, and projects in a clean, focused way.

---

## Features

- **Hero / About section**
  - Short introduction and a more detailed description of interests.
  - 3D style avatar used as the main visual focus.
  - Quick tags that highlight key roles and focus areas.

- **Experience timeline**
  - Vertical timeline from 2023 to present.
  - Separate cards for:
    - Data Analyst Intern at Shiash Info Solutions.
    - AI / ML Engineer at Tata Consultancy Services (TCS).
  - Bullet points that focus on practical work with data, dashboards, and AI assistants.

- **Skills**
  - Horizontally scrolling skill groups.
  - Each group shows a theme (Analytics, GenAI and NLP, Data and Cloud, Retrieval and Graphs, Apps and Prototyping) and tools used inside that area.

- **Projects**
  - Cards for company work, hackathon projects, academic work, and personal tools.
  - Each card shows:
    - Index number
    - Category (Company, Hackathon, Personal, Academic)
    - Title, short subtitle, tech stack, and a clearer description.

- **Contact**
  - Simple contact block with email and location.
  - Tags that show the kind of roles and work that are a good fit.

- **Responsive layout**
  - Layout adapts for desktop, tablet, and mobile.
  - Sections remain readable and scroll smoothly on different screen sizes.

---

## Tech stack

- **Frontend:** React (Vite)
- **Styling:** Custom CSS (no UI framework)
- **Language:** JavaScript (ESNext)
- **Build tooling:** Vite dev server and bundler

You can add extras such as Framer Motion or other libraries later if you want more animation.
