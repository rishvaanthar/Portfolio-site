# Rishvaanth Portfolio (React + Vite)

Single page portfolio similar in feel to moncy.dev, using your experience and projects.

## Requirements

- Node.js 18 or later
- npm

## Setup

```bash
npm install
npm run dev
```

Then open the URL that Vite prints (usually http://localhost:5173).

To build for production:

```bash
npm run build
npm run preview
```
